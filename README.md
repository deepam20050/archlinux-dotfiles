1. [DWM](https://dwm.suckless.org/)
2. [Neovim](https://neovim.io/)
3. [Fish Shell](https://fishshell.com/)
4. [Rofi](https://github.com/davatorium/rofi)
5. [Conqueror Of Completion](https://github.com/neoclide/coc.nvim)
6. [Qtile](http://www.qtile.org/)
7. [Sublime Text](https://www.sublimetext.com/)
